## Event Kind 30023 — Long-form Content

### Overview

Event Kind **30023** defines **long-form content** in Nostr.

- **Specification**: [NIP-23](https://github.com/nostr-protocol/nips/blob/master/23.md)
    
- **Replaceability**: Addressable event. Only the latest event per combination of `kind`, `pubkey`, and `d` tag is valid.

### Validation Rules

1. **Addressable** — addressable event rules apply.
    
2. **Specification requirement** — NOT hard line-break paragraphs of text, such as arbitrary line breaks at 80 column boundaries
    
3. **Specification requirement** — NOT support adding HTML to Markdown
    
4. **Specification requirement** — be made according to [NIP-27](27
