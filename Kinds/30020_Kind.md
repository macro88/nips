## Event Kind 30020 — Product sold as an auction

### Overview

Event Kind **30020** defines **product sold as an auction** in Nostr.

- **Specification**: [NIP-15](https://github.com/nostr-protocol/nips/blob/master/15.md)
    
- **Replaceability**: Addressable event. Only the latest event per combination of `kind`, `pubkey`, and `d` tag is valid.
