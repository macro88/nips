## Event Kind 10013 — Private event relay list

### Overview

Event Kind **10013** defines **private event relay list** in Nostr.

- **Specification**: [NIP-37](https://github.com/nostr-protocol/nips/blob/master/37.md)
    
- **Replaceability**: Follows [NIP-16](https://github.com/nostr-protocol/nips/blob/master/16.md). Only the most recent event per pubkey (by `created_at`) is valid.
